# 暗夜执行指令话术模板

> 用于获取实时股市数据的标准指令格式

---

## 标准指令模板

```
暗夜，我要获取 [股票代码/目标] 的实时数据。请你：

1. 利用 OpenCode 寻找一个稳定的数据源（如新浪、腾讯或百度财经）
2. 自主编写 Python 脚本进行抓取
3. 约束：优先使用轻量级的 requests 库；如果遇到复杂反爬，再切换到 playwright
4. 运行并直接告诉我结果
```

---

## 示例对比

### 以前的暗夜（无 OpenCode）
> "我无法获取实时金融数据，建议你去官网查看"

### 现在的暗夜（带 OpenCode）
> "我需要去抓数据"
> → 默默写脚本
> → 运行脚本
> → 返回结果

---

## 实战示例

**指令：** 
> "暗夜，帮我看看 600406（国电南瑞）今天的资金净流入情况"

**执行流程：**
1. 寻找数据源（东方财富/同花顺资金流向页）
2. 编写 Python 脚本（requests/playwright）
3. 运行脚本获取数据
4. 返回结果

**预期回复：**
> "执行官，国电南瑞今日主力净流入 XX 亿元，表现稳健。"

---

## 技术储备

| 工具 | 用途 |
|------|------|
| requests | 轻量级 HTTP 请求（首选）|
| playwright | 复杂反爬时使用 |
| OpenCode | Python 脚本执行环境 |

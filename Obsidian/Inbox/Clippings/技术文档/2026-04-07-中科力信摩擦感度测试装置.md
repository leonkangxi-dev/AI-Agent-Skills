# 北京中科力信预研新产品：基于ABL摩擦仪改进的多物理量测量的摩擦感度测试装置

## 来源链接
https://mp.weixin.qq.com/s/bqm4hqTGqXZkO_aK-Poihg

## 核心摘要
北京中科力信预研基于ABL摩擦仪改进的含能材料摩擦感度测试装置，用于测定固体、半固体或粉末状物质对机械摩擦刺激的敏感程度。

## 关键内容

### 主要功能
- 测定固体/半固体/粉末物质对机械摩擦刺激的敏感程度
- 判断在运输、加工过程中是否会发生燃烧、爆炸等危险反应
- 具备精细化、定量化摩擦感度自动化加载与多物理量测量能力

### 工作原理
将物料置于可移动砧座上，通过液压装置对上方压轮施加恒定正压力，利用摆锤冲击移动砧座做单次快速水平滑动，使试样在压轮与砧座间受到可控摩擦作用，通过传感器精确测量滑动摩擦力与垂直正压力。

### 可获取的物理量
- 滑动摩擦力
- 摩擦位移
- 摩擦功
- 垂直正压力

### 主要技术参数
| 参数 | 规格 |
|------|------|
| 规格尺寸 | 1.5m×1.0m×1.8m |
| 垂直压力测量范围 | 0~5000N, 精度1% |
| 低速驱动模式 | 0~1m/s |
| 高速驱动模式 | 1~2.5m/s |
| 摩擦板行程 | 50mm |

### 系统组成
1. **摩擦执行机构** - 固定轮、移动砧座、摆锤驱动系统
2. **压力加载系统** - 液压系统
3. **控制与测量系统** - 工控机及控制软件
4. **样品测试工装**

### 技术优势
- 实现摩擦速度、摩擦位移等基础量的自动测量
- 摩擦力、正压力精确测量
- 适配各种标准/非标准固定轮与移动砧座
- 支持客户定制

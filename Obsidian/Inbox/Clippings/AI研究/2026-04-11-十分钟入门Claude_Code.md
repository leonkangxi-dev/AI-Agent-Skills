## 第2页

Claude Code 是 Anthropic 公司（Claude AI 的制造商）推出的一个 AI 工作台。
你可以把它理解为一个住在你电脑里的超级助手：
它能读写你电脑上的文件——帮你整理文档、批量改名、分析数据
它能上网搜索——帮你做调研、找资料、追踪热点
它能安装各种技能包——做 PPT、生成图片、写推文、创建视频、分析 Excel
ChatGPT 像一个坐在屏幕里的顾问——你问它，它答你，但活还是你自己干。Claude
Code 像一个坐在你旁边的全能助手——你说一句，它直接帮你把事情做了，结果文件就
在你的电脑上。
在开始之前，你需要什么？
硬件： 一台能上网的电脑（Mac、Windows、Linux 都行）
软件： 不需要提前装任何东西，下面会一步步教你
费用： Claude Code 需要订阅 Claude Pro（每月 $20）或 Claude Max（每月
$100）。Pro 计划足够日常使用。如果你还没有
Claude 账号，去 claude.ai 注册一个。
技术基础： 零。真的是零。你只要会打字就行。
第一步：安装 CLAUDE CODE（3 分钟）
方法一：桌面应用（最简单，推荐新手）
Claude Code 现在有桌面应用了，安装跟装微信一样简单：
打开 claude.ai/code
下载对应你系统的安装包（Mac 或 Windows）
双击安装，打开应用

## 第3页

用你的 Claude 账号登录
搞定。没有命令行，没有配置，开箱即用。
方法二：命令行安装（适合想多了解一点的人）
如果你想用命令行版本（功能完全一样，只是界面不同），需要先装一个叫 Node.js
的东西。别怕，跟着做就行：
第一步：安装 Node.js
去 nodejs.org 下载"LTS"版本（左边那个绿色按钮），双击安装，一路点"下一步"就行。
第二步：安装 Claude Code
Mac 用户：打开"终端"（在启动台搜索"终端"或"Terminal"）
Windows 用户：打开"命令提示符"或"PowerShell"（按 Win 键搜索即可）
输入这一行命令，然后按回车：
npm install -g @anthropic-ai/claude-code
等它跑完（通常不到一分钟），再输入：
claude
第一次运行会让你登录 Claude 账号，按提示操作就行。
看到欢迎界面？恭喜，你已经装好了。
方法三：网页版（零安装）
如果你连安装都不想搞，可以直接在浏览器里用：
打开 claude.ai/code
选择 "Web" 版本

## 第4页

直接开始用
网页版功能会少一些（比如不能直接操作你电脑上的文件），但用来体验和学习完全足够
。
打开 Claude Code 后，你会看到一个输入框。就像跟朋友发消息一样，直接打字就行。
试试这个：
帮我写一篇 500 字的自我介绍，我叫张三，做跨境电商 5 年了，擅长选品和供应链管理。
风格要专业但不要太正式。
按回车，等几秒钟。
你会看到 Claude Code
直接帮你写好了一篇自我介绍——而且它不是那种一眼就能看出是 AI
写的八股文，而是会根据你给的信息来定制内容。
恭喜，你已经在用 Claude Code 了。
再试一个：
帮我列一个表格，对比亚马逊、Shopify 和 TikTok Shop 三个平台，
从入驻门槛、费用结构、适合品类、流量特点四个维度来分析。
结果保存为一个 Markdown 文件。
这次 Claude Code
不但帮你做了分析，还把结果保存成了一个文件。你可以在你的电脑上直接打开它。
这就是 Claude Code 和普通 AI
聊天工具的核心区别——它能直接操作文件，帮你把成果落地。

## 第6页

帮我做三件事：
1. 把上周写的产品方案翻译成英文
2. 基于这个方案生成一份 10 页的投资人 PPT
3. 再写一条 200 字以内的产品推文
一个指令，三个产出。以前这三件事你可能要在三个不同的工具里分别做。
五个让你事半功倍的技巧
1. 给它足够的上下文
差的指令：
"帮我写一篇文章"
好的指令：
"帮我写一篇 2000
字的公众号文章，主题是中小卖家如何在亚马逊上打造差异化。读者是做了 1-3
年跨境电商的人。语气要务实、接地气，多用具体案例。"
你告诉它的信息越多，产出越精准。
2. 让它参考你的风格
如果你之前写过文章或文档，可以让 Claude Code 先读一下：
先读一下"我的文章"文件夹里的几篇文章，学习我的写作风格，
然后用同样的风格帮我写一篇新文章。
它会模仿你的用词习惯、句式结构、论证方式。写出来的东西，别人读了会觉得就是你写
的。
3. 用 /clear 保持清醒

## 🎯 n8n 核心认知

### 什么是 n8n？
- **开源**工作流自动化平台
- **400+** 内置集成
- 支持**自托管** (数据自主)
- 可视化拖拽编辑器
- 原生支持 AI/LLM 集成

### 与 Zapier/Make 的区别
| 特性 | n8n | Zapier/Make |
|------|-----|-------------|
| 开源 | ✅ | ❌ |
| 自托管 | ✅ | ❌ |
| 数据控制 | 完全自主 | 依赖第三方 |
| 成本 | 免费自托管/免费云 tier | 按任务付费 |
| AI原生支持 | ✅ 内置 | 有限 |

## 💡 核心学习洞察

### 1. 工作流设计模式
```
触发器 (Webhook/Schedule/Event)
    ↓
数据处理 (Transform/Filter)
    ↓
AI处理 (LLM/Agent/Vision)
    ↓
数据存储 (DB/Vector Store)
    ↓
通知/输出 (Slack/Email/Telegram)
```

### 2. AI集成最佳实践
- **从简单开始**: 裸调API → 简单Agent → 复杂工作流
- **Human-in-the-loop**: 关键环节人工确认
- **本地优先**: Ollama等本地模型降低成本
- **RAG增强**: 向量数据库提升回答质量

### 3. 高频应用场景
| 场景 | 技术组合 | 价值 |
|------|----------|------|
| 智能客服 | Telegram/Slack + RAG + LLM | 24/7自动回复 |
| 内容生成 | WordPress/社交媒体 + AI | 批量创作 |
| 数据分析 | 数据库 + Text-to-SQL + 可视化 | 自然语言分析 |
| 文档处理 | PDF + OCR + 结构化提取 | 自动化录入 |
| 竞品监控 | 爬虫 + AI摘要 + 通知 | 实时情报 |

## 💡 工具选择策略

### 按场景推荐

| 场景 | 推荐工具组合 |
|------|-------------|
| **日常办公** | Notion AI + ChatGPT + Perplexity |
| **内容创作** | Claude + Midjourney + ElevenLabs |
| **编程开发** | GitHub Copilot + Claude Code + Phind |

| **客户服务** | SiteGPT + Intercom Fin |

### 成本优化策略

**免费替代方案**
- ChatGPT Plus → Claude Free + Perplexity Free
- Midjourney → Stable Diffusion (本地)
- ElevenLabs → Bark (开源)
- GitHub Copilot → Codeium (免费)

**混合策略**
- 主力工具付费(确保质量)
- 辅助工具用免费/开源
- 批量任务用API(成本低)

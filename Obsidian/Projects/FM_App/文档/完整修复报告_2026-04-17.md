## 修复清单

### Phase 1: 播放器核心修复 ✅

#### 1. 循环切换逻辑
**问题**: prev/next 到边界停止，不循环
```javascript
// 修复前
function prevChannel() {
    if (currentIndex > 0) playChannel(currentIndex - 1);  // 到0就停
}
function nextChannel() {
    if (currentIndex < channels.length - 1) playChannel(currentIndex + 1);  // 到最后就停
}
```

**修复后**: 循环导航 + 自动跳过失效频道
```javascript
function prevChannel() {
    if (channels.length === 0) return;
    const index = currentIndex <= 0 ? channels.length - 1 : currentIndex - 1;
    playChannel(index);
}

function nextChannel() {
    if (channels.length === 0) return;
    let attempts = 0;
    let index = currentIndex;
    do {
        index = index >= channels.length - 1 ? 0 : index + 1;
        attempts++;
    } while (failedChannels.has(channels[index]?.id) && attempts < channels.length);
    
    if (attempts < channels.length) {
        playChannel(index);
    }
}
```

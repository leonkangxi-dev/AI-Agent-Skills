# FM App 知识库 - 技术文档

## 项目概述
网络电台播放器，支持 CCTV、凤凰卫视、RTHK 等频道。

## 架构

### 前端
- `radio-player.html` - 主播放器 (Taste Skill 设计)
- `admin/index.html` - 管理员登录
- `admin/dashboard.html` - 管理后台

### 后端
- Flask + SQLite
- CORS 支持
- RESTful API

### 数据库 Schema

```sql
-- 频道表
CREATE TABLE channels (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    url TEXT NOT NULL,
    group_name TEXT DEFAULT 'General'
);

-- 管理员表
CREATE TABLE admins (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT UNIQUE NOT NULL,
    password TEXT NOT NULL
);
```

## API 文档

### 频道管理
```
GET    /api/channels              # 列表
POST   /api/channels              # 创建
PUT    /api/channels/:id          # 更新
DELETE /api/channels/:id          # 删除
```

### 管理员
```
POST /api/admin/login             # 登录
GET  /api/admin/admins            # 列表
POST /api/admin/admins            # 创建
DELETE /api/admin/admins/:id      # 删除
```

## 设计规范

### 颜色
- 主色: #0284c7 (Sky 600)
- 背景: 蓝白渐变
- 毛玻璃: rgba(255,255,255,0.75)

### 字体
- 标题: Geist
- 正文: Inter

### 动画
- 脉冲: 4s 周期
- 按钮: 物理压感 (translateY + scale)

## 常见问题

### M3U8 无法播放
- 浏览器 CORS 限制
- 解决方案: 使用代理服务器

### 自动跳过失效频道
播放器会记录失效频道 ID，自动跳过

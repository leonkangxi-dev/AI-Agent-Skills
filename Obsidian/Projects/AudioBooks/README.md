# AudioBooks 项目

> 电子书语音化转换与管理

## 目录结构

```
AudioBooks/
├── 📖 工作流文档/
│   └── 电子书语音书转换工作流.md    ← 完整操作手册
└── 📚 有声书笔记/
    └── *.md                       ← 各书籍的转换记录
```

## 使用方法

通过 Telegram 发送电子书文件（PDF / EPUB / MOBI / AZW3 / TXT）给 Bot，自动触发转换流程。

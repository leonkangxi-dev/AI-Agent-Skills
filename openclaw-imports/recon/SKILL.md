# Recon 网络侦察系统

当需要探测系统状态、扫描网络服务、检测可用端口时使用此 skill。

## 功能列表

### 1. 端口扫描
检测指定IP的开放端口
```
使用方法：scan_ports <IP> [端口范围]
示例：scan_ports 192.168.1.1 1-1000
```

### 2. 服务检测
检测本地服务运行状态
```
使用方法：check_services
检测：3000, 5000, 5001, 18789, 9996-9998
```

### 3. 网络探测
探测网络连通性和延迟
```
使用方法：ping <目标>
示例：ping google.com
```

### 4. 设备发现
扫描局域网内设备
```
使用方法：scan_network [网段]
示例：scan_network 192.168.1
```

### 5. 服务健康检查
批量检测服务可用性
```
使用方法：health_check
```

## 常用命令

| 命令 | 功能 |
|------|------|
| scan_ports | 端口扫描 |
| check_services | 服务状态检测 |
| ping | 网络连通性 |
| scan_network | 局域网设备发现 |
| health_check | 批量健康检查 |

## 输出格式

检测结果以表格形式展示：
- 服务名
- 端口
- 状态 (UP/DOWN)
- 响应时间

## 自动触发

当用户提及"检查网络"、"看看有哪些服务"、"端口扫描"等关键词时，自动调用相应功能。
